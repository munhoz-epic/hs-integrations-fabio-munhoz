# HubSpot Integration Fabio Munhoz Practicum

## Setup Instructions

1. Clone this repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file in the root directory with the following variables:
   ```
   HUBSPOT_API_KEY=
   CUSTOM_OBJECT_TYPE=
   ```
4. Start the server:
   ```bash
   npm start
   ```
5. `http://localhost:3000`
